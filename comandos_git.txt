git init. -> cria um repo git na pasta atual
git status -> mostra o status
git add nome_arquivo -> adiciona arquvio pra ser comitado
git commit -m "msg" -> faz commit do que estava para ser comitado
git log -> mostra todo o historico de commit

## Fluxo de trabalho
1. criar o diretorio 
2. iniciar o git dentro do diretorio (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome do arquivo>
6. git status
7. git commit -m 'mensagem'
8. git status
9. volta passo 3
